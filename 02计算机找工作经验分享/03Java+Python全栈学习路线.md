# Java+Python全栈学习路线

## 先纠正一个误区

纠正一下很多小白的误区，Java、Python 其实不是两条分开的道路，所以不要再问：**学 Java，还是学 Python** 这种话了。

现在岗位统一叫作 **Agent 开发工程师**，岗位需求就是前后端全是一个人干，后端开发可能是传统项目的 Java 或者 Python 的大模型应用开发。如果只会大模型应用开发，学历不够，会死的很惨。

**我的建议就是都学。**

原因很简单：

- 现在前后端基本不分家
- 传统业务以 Java 为主，存量项目大量需要维护；同时企业有大模型落地需求，Python 做 AI 应用开发更顺手

## 学习策略

- 先速成跑通全流程，别一开始就钻牛角尖
- 后续根据简历、面试需要再体系化补八股
- 全程用 AI 辅助学习（Claude / ChatGPT），做项目的时候学一下怎么使用编程助手来帮自己看懂某个接口

## 四个学习阶段

> **Java 后端基础 → 前端基础 → AI 应用开发 → 工程化能力**

### 第一阶段（约 2 个月）：Java 后端基础

直接看黑马程序员：https://www.bilibili.com/opus/494817843530680807

### 第二阶段（约 1 周）：补前端，具备完整项目能力

推荐黑马程序员前端速成课 `BV1kM4y127Li`

> 💡 到这里其实就可以开始投实习了。尽早积累实战经历。
> 学历一般的同学路径：**小厂 → 中厂 → 大厂**，一步一步打怪升级。
> 此时简历上是：魔改的点评 + 外卖

### 第三阶段（约 1 个月）：Python + AI 应用开发

核心学 **LangGraph**。学完之后用 LangGraph 框架 + Claude Code / Codex，自己手敲一个完整项目，把高级特性都用一遍。

到这个阶段基本就领先很多人了。直接看我小红书笔记：`5332538440`

> 此时简历上是：一段包装实习经历 + LangGraph 项目

### 第四阶段：工程化能力

Linux、Docker、云服务器部署。每个项目都要考虑：

- 并发处理
- 限流策略
- 安全防护
- 日志 & 监控
- 接口性能测试
- 接口效果评估

这部分是从"能写代码"到"能上线项目"的关键跨越，也是需要一直优化的点，可以购买一些优秀项目进行学习。

> 此时简历上是：优化后的实习经历 + 项目
