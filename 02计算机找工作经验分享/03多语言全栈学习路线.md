# 多语言全栈学习路线

## 先纠正一个误区

纠正一下很多小白的误区，Java、Python 其实不是两条分开的道路，所以不要再问：**学 Java，还是学 Python** 这种话了。

现在岗位统一叫作 **Agent 开发工程师**，岗位需求就是前后端全是一个人干，后端开发可能是传统项目的 Java 或者 Python 的大模型应用开发。

我的建议就是**不要划分语言了，都学。**

原因很简单：

- 现在前后端基本不分家
- 传统业务以 Java 为主，存量项目大量需要维护；同时企业有大模型落地需求，Python 做 AI 应用开发更顺手
- 相对于大模型应用开发，Java的学习资料非常完善，有助于你提升编程理解力

## 学习策略

- 先速成跑通全流程，别一开始就钻牛角尖
- 后续根据简历、面试需要再体系化补八股
- 全程用 AI 辅助学习（Claude / ChatGPT），做项目的时候学一下怎么使用编程助手来帮自己看懂某个接口

### 第四阶段：工程化能力

Linux、Docker、云服务器部署。每个项目都要考虑：

- 并发处理
- 限流策略
- 安全防护
- 日志 & 监控
- 接口性能测试
- 接口效果评估

这部分是从"能写代码"到"能上线项目"的关键跨越，也是需要一直优化的点，可以购买一些项目进行学习，
去看Github开源项目，最后自己手撸一个完整项目

> 此时简历上是：优化后的实习经历 + 项目

### 八股怎么学

不要上来就死背，分两步走：

1. **先过一遍，理清大概的逻辑**：每个知识点画个图，搞清楚"是什么、为什么、怎么用"
2. **再背八股**：有了理解之后再背，记得牢，面试时也能扩展着讲
