## 使用技巧
- 在当前目录下配置CLAUDE.md（claude）或者AGENTS.md（codex），每次运行前会先默认加载这个文件的内容<br>
![img.png](../image/使用技巧.png)
- 配置，直接问AI配置文件在个目录，点进去修改即可，这样就不会每次修改文件前来问你，去掉这个烦人的步骤
