# 学习资源

## 先说结论

**先初阶入门，写完简历再定自己该学哪些，而不是乱学一通。** 简历上需要什么，就去补什么，目标明确才不会浪费时间。

---

## 四个阶段对应不同的学习资源

### 初阶：小白刚入门

看 B 站黑马程序员、尚硅谷免费网课就够了，不需要花钱。

这个阶段的目标：**跑通一个完整项目**，从前端到后端到数据库，知道一个系统是怎么跑起来的。

### 进阶：做过几个项目，基本逻辑都理清楚了

这个阶段不要再只看视频了，学习方式要升级：

- **官方文档 + AI 辅助**：加快学习常见中间件（Redis、MQ、ES 等）
- **买好项目学习**：B 站的项目太玩具了，得买一些教学完善、好评多的付费项目

> 我进的是马丁的知识星球，虽然烂大街，但项目质量和教学质量确实不错。拿到项目后发给 AI 魔改一下就好了。

其他推荐：

| 推荐 | 说明 |
|------|------|
| JavaGuide | 八股文 + 学习路线，面试必备 |
| 小傅哥 | 项目质量不错，适合进阶 |
| 鱼皮 | 项目课程，适合跟着做 |

### 高阶：能独立做出能上线的项目

- 去 **GitHub** 上找优秀开源项目，阅读源码
- 能 vibe coding 出一个完整的、能打的项目
- 学习资料：**官方文档 + 源码阅读 + 经典书籍**

### 天阶：给框架提 PR，手搓框架

到这个阶段已经不需要别人推荐资源了，你自己就是资源。
