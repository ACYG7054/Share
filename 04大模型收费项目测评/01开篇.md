## 评估维度

每个项目/课程/社区，我会从以下维度进行打分（满分 5 星）：

| 维度         | 说明                           |
|------------|------------------------------|
| **核心代码行数** | 是否是屎山代码堆叠                    |
| **上手难度**   | 对新手是否友好，文档和视频是否清晰            |
| **教学质量**   | 讲解是否到位，是否有循序渐进的引导，而不是一上来就堆代码 |
| **项目质量**   | 代码规范、架构设计、技术栈是否主流，能不能写进简历    |
| **性价比**    | 价格和内容是否匹配，值不值这个钱             |
| **售后/社区**  | 有没有答疑、更新维护、社群交流              |

### 评估指标

```
核心代码行数：4w行
上手难度：⭐⭐☆☆☆（较难）
教学质量：⭐⭐⭐⭐☆（优秀）

性价比：⭐⭐⭐☆☆（中等）
售后：⭐⭐⭐☆☆（中等）
```
